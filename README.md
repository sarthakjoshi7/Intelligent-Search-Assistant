# Intelligent Search Assistant

An AI powered search application built using LangChain, Open Source LLM, agents and tools.

## Project Goal

This project demonstrates the implementation of an intelligent search assistant that utilizes multiple external tools and AI agent reasoning to retrieve, process, and present information from the web.

## Features

- Search the web using search engines
- Retrieve information from Wikipedia
- Access research papers through Arxiv
- AI agent based reasoning and tool selection
- Real time information retrieval
- Conversational chat interface
- User interface

## Tech Stack

- Python
- Streamlit
- LangChain
- Open Source LLM API
- Search Engine
- Llama 3.1
- Agents
- Tools

End of project.
